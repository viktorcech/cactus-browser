VBXE Web Browser

Welcome! You are browsing the web on a real Atari XL/XE computer with a VBXE video expansion and FujiNet network adapter.

This is an 8-bit web browser written in 6502 assembly language. It renders HTML pages in 80-column text mode using the VBXE overlay.

Features

  • 80-column text - VBXE text overlay mode (TMON)
  • Color rendering - headings, links, bold text in color
  • Image viewing - fullscreen 256-color images via VBXE graphics overlay
  • Mouse support - ST mouse for clicking links
  • Word wrapping - automatic line breaking at 80 columns
  • HTML tags - headings, paragraphs, links, lists, tables, bold, italic, HR
  • Page buffering - entire page downloaded to VRAM before rendering
  • Link navigation - click links or use keyboard

Controls

Navigation

  • U - enter a new URL
  • B - go back to previous page
  • Q - quit current page (return to start screen)

Page scrolling

  • Space or Return - scroll to next page
  • H - skip to next heading (useful for skipping navigation menus)
  • Click on --More-- bar - same as Space

Links and images

  • Click on a link - follow the link
  • Click on [IMG] - download and show image fullscreen
  • Press any key to close image and return to page

Requirements

  • Atari XL/XE - 64KB RAM
  • VBXE - Video Board XE (FX core 1.x)
  • FujiNet - WiFi network adapter
  • ST Mouse - Atari ST mouse in port 2

Technical details

The browser is written entirely in 6502 assembly using the MADS (Mad Assembler) assembler. It uses approximately 10 KB of code.

  • Display - VBXE XDL with TMON overlay, 80x29 characters
  • Network - FujiNet N: device via SIO ($71)
  • Images - server-side conversion to 320x184, 256-color indexed format
  • Page buffer - HTML stored in VRAM ($14000+) for offline rendering
  • Palette - 8 text colors + 64 blue shades for link detection
